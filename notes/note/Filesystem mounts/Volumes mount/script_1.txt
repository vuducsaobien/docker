3.1 Run bằng --mount 
     docker run -d \
    --name devtest \
    --mount source=myvol2,target=/app \
    nginx:latest

3.2 Run bằng -v
     docker run -d \
     --name devtest \
     -v myvol2:/app \
     nginx:latest

5. Kiểm tra Chi tiết 1 Container
     Use docker inspect devtest to verify that Docker created the volume and it mounted correctly. 
     Look for the Mounts section:

     - docker inspect devtest
          ....................................
          "Mounts": [
                         {
                              "Type": "volume",
                              "Name": "myvol2",
                              "Source": "/var/lib/docker/volumes/myvol2/_data",
                              "Destination": "/app",
                              "Driver": "local",
                              "Mode": "z",
                              "RW": true,
                              "Propagation": ""
                         }
                    ],
          ....................................
     This shows that the mount is a volume, it shows the correct source and destination, and that the mount is read-write.

6. Stop the container and remove the volume. Note volume removal is a separate step.
     docker container stop devtest
     docker container rm devtest
     docker volume rm myvol2
